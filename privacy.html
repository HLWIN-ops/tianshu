天枢 · 信任中心返回排盘

隐私说明

生效日期：2026 年 7 月 16 日。天枢的默认原则很简单：完成一次排盘，不需要把出生资料交给我们。

一句话版本：排盘在当前浏览器完成；不要求注册；只有你主动保存时，资料才进入本机档案。档案和导出文件都可能包含敏感出生信息，请像保管私人文件一样保管它们。

1. 页面会处理哪些资料

为了完成排盘，页面会读取你填写的姓名或本地称呼、传统排运性别、出生日期、时间、城市、经度、时区、历法与规则选项。其中姓名可以不填。页面代码不需要把这些字段发送给应用服务器。

2. 本机保存并不等于加密

  • 主动点击“保存到档案”，或首次点击“保存本月记录”后，命盘资料会以明文 JSON 保存在当前浏览器的 localStorage 中。
  • 本月记录还会明文保存行动、进度、月份和你填写的“现实证据”自由文本，并通过随机本机档案 ID 关联；关联 ID 本身不包含姓名、生辰或经纬度。
  • 能使用这台设备和该浏览器配置的人，可能读取这些资料；请勿在公共或不受信任的设备上保存。
  • 清理浏览器数据、使用无痕窗口、重装浏览器或更换设备，都可能让档案永久丢失。
  • 导出的 JSON 含完整出生资料且没有密码保护。请勿上传到公开网盘、群聊或代码仓库。

3. 分享默认脱敏

天枢生成的默认文本摘要和人物相似度海报，不包含姓名、出生日期、出生时间、地点和完整四柱。分享前会显示脱敏预览；海报二维码只在本机编码去除查询参数和片段后的当前站点地址，不包含命盘数据。分享内容仍会包含日主、格局、年度主题或人物匹配等推导结果；如果你认为这些结果也属于隐私，请不要分享。手动截图不受自动脱敏机制保护。

4. 网络、统计与托管边界

当前项目代码没有账号、广告、支付、远程 AI、第三方统计脚本或出生资料上传接口。静态站点由部署者选择的托管平台提供，平台可能按其规则处理访问日志、IP 地址和安全事件；这些基础托管日志不包含页面表单字段。

如果未来启用 Cloudflare Web Analytics 或其他统计能力，必须先更新本说明，并坚持不把姓名、出生日期、时间、地点、四柱或档案标识写入统计事件。

5. 如何删除自己的资料

进入“档案”可逐份删除或清空全部档案；删除档案时会同时删除其行动记录。结果页也可只清除当月记录。你还可以在浏览器的网站数据设置中清除此站点的 localStorage。已下载或自行分享的文件，需要由你在对应设备或平台另行删除。

6. 产品边界

请不要录入未获授权的他人资料。天枢不会以恐惧话术诱导付费，也不会宣称能诊断疾病、保证收益、确定关系结果或替代专业意见。发现隐私风险时，应暂停保存和分享，并通过 项目公开问题渠道反馈。
